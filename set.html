set:数据结构 -》没有重复的值

API

var set = new Set([1,2,3,4,5])
1.set.add(value):添加值，可链式添加
2.set.delete(value):删除值，返回布尔值，表示成功失败
3.set.has(value):判断是否包含某个值，返回布尔值
4.set.clear():清楚所有数据，没有返回值
5.set.keys(),set.values():访问建名和值，set建和值都是一样的。 7.set.entries():返回键值对遍历器{【1,1】，【2,2】。。。。} 8.set.forEach(function(value,key,set){})数据结构本身没有变化。
